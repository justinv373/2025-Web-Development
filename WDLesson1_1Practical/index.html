Web Technologies

HTML

HTML, or HyperText Markup Language, is the standard markup language used to create and structure content on the World Wide Web. It uses a system of "tags"
to define elements like headings, paragraphs, images, and links, dictating how a web browser should display these components. Essentially, HTML provides the
foundational structure and meaning for all web pages, forming the backbone of what users see online.

CSS

CSS, or Cascading Style Sheets, is a style sheet language used for describing the presentation of a document written in a markup language like HTML. It allows
web developers to control the visual appearance of web pages, including elements such as layout, colors, fonts, and responsiveness across different devices.

JavaScript

JavaScript is a programming language used primarily to add interactive behavior to web pages. It works alongside HTML and CSS, which handles a website's
structure and appearance, by allowing developers to create animations, validate forms, and update content in real time.
